Segment.io

Просматривая фид dribbble, нередко натыкаешься на шоты, где пользователи презентуют свои проекты. Так я наткнулся на Segment.io, как всегда я решил посмотреть что за штука такая. Тогда они предоставляли api только на js для client-side, поставил на бложик, и их простота мне приглянулась. Основная идея — объединить сервисы, собирающие статистику и унифицировать предоставляемый ими api.

Сейчас у них уже есть модуль для NodeJS, описание REST API и очень скоро появится gem для ruby.

Список поддерживаемых «аналитик» на момент написания поста:

Код на js для подключения лежит на github и к нему написана хорошая документация.

Судя по модулю для NodeJS, работают они давно, шли медленно и не спеша, что не может не вдохновлять, ибо я тоже иду очень медленно. Я спрашивал у разработчиков что будет дальше, сказали что если они и станут платными, то только при использовании интеграции со сторонними сервисами, т.е. их собственная статистика должна быть бесплатной. А еще мне обещали вечный «pro» аккаунт, в качестве награды за участие в раннем бета-тестировании :3
